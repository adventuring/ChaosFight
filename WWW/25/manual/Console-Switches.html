8 Console Switches ¶

The console switches provide additional game control:

  • GAME RESET Switch
  • GAME SELECT Switch
  • TV TYPE Switch (COLOR/B&W)
  • DIFFICULTY Switches

8.1 GAME RESET Switch ¶

Pressing the RESET switch immediately starts the game over.

8.2 GAME SELECT Switch ¶

Pressing the SELECT switch:

  1. Re-detects controller configuration
  2. Pauses the game

Note, you cannot remove a Quadtari and re-detect from within the game; perform detection on the Title or Character Select screens.

8.3 TV TYPE Switch (COLOR/B&W) ¶

On NTSC or PAL systems, when in "COLOR" mode, the playfield will be colorful.

On SECAM systems, or when in "B&W" mode on NTSC or PAL, the playfield will be white on black.

On a 7800, press Pause to toggle between color and black & white modes.15

Characters are always colored based on player number.

8.4 DIFFICULTY Switches ¶

The RIGHT DIFFICULTY switch in A (Advanced) position will cause the CPU player to play at full health in a 1-player game. In the B (Novice) position, the CPU player will play handicapped (with 75% health)

The LEFT DIFFICULTY switch is NOT used.

Footnotes

(15)

On Atari 7800, the pause button occupies the same hardware pin as the Color/B&W switch, so it toggles the playfield color mode instead of using the switch position.
