3 Controller Configuration ¶

ChaosFight supports multiple controller types with automatic detection. Button mappings vary by controller type.

  • Standard joystick controllers
  • Genesis/Mega Drive Controllers
  • Joy2b+ Enhanced Controllers
  • Four-Player Mode with Quadtari

3.1 Standard joystick controllers ¶

  • LEFT/RIGHT Move and turn character
  • UP Jump
  • DOWN Guard
  • Fire Button Attack. Also used for character and arena selection.

Plug Player 1 into left port, Player 2 into right port.

3.2 Genesis/Mega Drive Controllers ¶

Genesis/Mega Drive controllers supported with automatic detection.

  • Button B (Middle) Attack. Also used for character and arena selection.
  • Button C (Right) Jump (alternative to UP)
  • Buttons A and START Not usable (hardware limitation)

Only button B will work when connected through a Quadtari (just like a standard joystick)

3.3 Joy2b+ Enhanced Controllers ¶

Joy2B+ controllers are modern enhanced joysticks designed for the Atari Video Computer System, featuring three fire buttons

  • LEFT/RIGHT: Move and turn your fighter
  • DOWN: Guard
  • UP: Jump
  • Button I Attack. Also used for character and arena selection.
  • Button II Jump - Alternative to UP
  • Button III Pause - Alternative to Game Select

Important Note:

Like Genesis/Mega Drive controllers, Joy2B+ controllers’ additional buttons do not work via Quadtari; only Button I will work.

3.4 Four-Player Mode with Quadtari ¶

Quadtari will be detected automatically if you have it attached when you power on your system. If not, press Game Select on the title screen or Character Select screen to re-detect.

The Quadtari must be connected to both Left and Right controller ports in order to be detected properly.
