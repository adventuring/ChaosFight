7 Game Modes ¶

ChaosFight offers multiple ways to play:

  • Two-Player Battle
  • Four-Player Chaos
  • Battle Arenas

7.1 Two-Player Battle ¶

Classic one-on-one combat. Each player selects a character and fights until one is victorious.

7.2 Four-Player Chaos ¶

Requires Quadtari adapter. All four players battle simultaneously in a free-for-all! Last player standing wins.

7.3 Battle Arenas ¶

Select from 2 different arenas during Arena Select! More arenas coming soon. Each arena offers unique tactical opportunities.

  • Arena Selection

7.3.1 Arena Selection ¶

After character selection, Player 1 can choose an arena:

  • LEFT/RIGHT Cycle through arena numbers (or "??" for random arena)
  • FIRE/B/I button Start the game with selected arena

Other players can:

  • Hold FIRE/B/I button for 1 second Return to character selection screen
  • Press GAME SELECT Return to character selection screen
