## üìã Format for Forum Topic URLs

AtariAge forum topics use this format:
```
https://forums.atariage.com/topic/[TOPIC_ID]-[slug-title]/
```

Example:
- Topic ID: 345295
- Slug: "bernie-the-tower-of-doom-done"
- Full URL: `https://forums.atariage.com/topic/345295-bernie-the-tower-of-doom-done/`

## üìã Format for Steam App URLs

Steam app URLs use this format:
```
https://store.steampowered.com/app/[APP_ID]/[Game-Name]/
```

Example:
- App ID: 413150 (for Stardew Valley)
- Full URL: `https://store.steampowered.com/app/413150/Stardew_Valley/`
